О себе

Проявляю интерес в изучении анализа и визуализации данных со времен распространенности Excel и написания макросов на VBA. После начала изучения Power Platform переключился на инструменты и платформы для анализа и визуализации данных. На данный момент работаю на платформе Power BI, но для общего развития практикуюсь в Tableau, а также на различных open-source инструментах, как Metabase, Apache Superset, Redash. Для получения данных по Rest API зачастую использую Python, знаком со скриптингом на Google Apps Script (для более полного взаимодействия с Google Sheets). Для реализации Pet проектов использую Docker. Очень интересно всё, что связано с технически сложными проектами по анализу данных, основной объем работ выполняю по описательной и диагностической аналитике, но часть задач развиваю в направлении предиктивной аналитики.
